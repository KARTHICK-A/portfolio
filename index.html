Cold Chamber Monitoring System

Delivered — client project

A 3-node distributed sensor network monitoring cold chamber temperature profiles from +50 °C down to −20 °C. Node 1 acquires sensor data, Node 2 stores and relays it, Node 3 bridges to a PC with Python software I wrote. Commands are acknowledged with a 3-retry protocol, and sessions survive power cycles.

ESP32RS485 half-duplexLoRa E32 (1–10 km)SPIFFS loggingCommand/ACK protocolPython PC software
