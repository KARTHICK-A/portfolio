/01 — PROJECTS

Built and shipped

2025–2026 · DELIVERED TO DELPHI TVS

Cold Chamber Monitoring System

A 3-node distributed sensor network monitoring cold-chamber temperature profiles from +50 °C to −20 °C. Node 1 acquires sensor data, Node 2 stores and relays, Node 3 bridges to a PC over custom Python software. Commands run on a 3-retry ACK protocol; sessions persist across power cycles.

  • ESP32
  • RS485 half-duplex
  • LoRa E32
  • SPIFFS logging
  • Command/ACK protocol
  • Python

2024–PRESENT · TESTED LIVE ON A MARINE ENGINE

Dual-Fuel Engine Control Unit (ECU)

A low-cost, customizable ECU for dual-fuel IC engine conversion. Reads MAP, engine speed, and exhaust gas temperature; sensor calibration and integration done in-house across multiple firmware versions.

  • STM32
  • ESP32
  • ADS1115
  • MAP/EGT sensors
  • CAN

2024–PRESENT · V2 IN MARKET TESTING

Oil Control Unit (OCU)

Electronic oil-pump controller with a MOSFET/TIP122 power stage and PWM control. Custom PCB designed, manufactured, and tested; version 2 is in market testing.

  • ESP8266
  • MOSFET/TIP122
  • PWM
  • Custom PCB

POC FOR INDO TECH TRANSFORMERS LTD

Industrial Transformer Monitoring System

Industrial IoT monitoring for transformers: pressure, voltage, current, temperature, and running hours streamed to the cloud and displayed on a P10 LED panel, with local SD logging as fallback.

  • RTC + SD
  • HTTP/JSON
  • P10 display
